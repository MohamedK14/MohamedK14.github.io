01 Real-Time Aircraft Tracking System Featured Full Stack
PythonRTL-SDRLeaflet
▾

Built a real-time aircraft monitoring system using an RTL-SDR receiver and a 1090 MHz antenna to capture ADS-B transmissions from overhead planes. A Python pipeline (dump1090) decodes, filters, and stores live telemetry — position, altitude, speed, and identification — into a SQLite database, with ~65% of messages carrying valid positional data and updates every 1–2 seconds. Data is served to an interactive Leaflet map with live tracking, historical trip visualization, altitude-colored paths, and unsupervised clustering to detect climb, cruise, and descent phases. An end-to-end project spanning hardware signal capture, real-time data pipelines, and ML — built to understand how systems talk to each other at every layer.
